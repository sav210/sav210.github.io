Хранение и отслеживание данных

Какие данные мы сохраняем?

  • Роли, каналы и их настройки;
  • ID наказанных пользователей;
  • ID серверов.

Для каких целей мы сохраняем выше перечисленные данные?

  • Для восстановления сервера;
  • Для сохранения наказаний до их отмены;
  • Для сохранения настроек бота.

Какие данные отслеживаются?

  • Добавление и удаление бота с серверов.

Правила

Запрещается:

  • Фальсификация бота (создание ботов с таким же названием и аватаром);
  • Фальсификация личности разработчика бота;
  • Использование неисправностей бота с целью причинить ущерб чему-либо;
  • Полное копирование бота;
  • Использование бота на серверах, посвящённым рейдам;
  • Злоупотребление командами бота.

В случае нарушения правил пользователь будет добавлен в чёрный список, ограничивающий использование бота.
